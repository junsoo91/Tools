🏘 주민등록인구 현황 차트

행정안전부 jumin.mois.go.kr · 행정동별 5세별 연간 데이터
📋 데이터 수집 가이드 (새 연도 추가 방법) ▼

① 사이트 접속

https://jumin.mois.go.kr/ageStatMonth.do

② 조회 조건 설정

  1. 조회기간 → "연간" 라디오 버튼 선택 (기본값 월간 → 반드시 변경)
  2. 시작연도 / 종료연도 선택 — 최대 3년 구간 (예: 2026 ~ 2028)
  3. 연령 구분 단위 → "5세" 선택 (기본값 10세 → 반드시 변경)
  4. 연령 조회 범위 → 좌측 0, 우측 100 선택 (기본 0~4세만 표시 → 반드시 변경)

③ 검색 및 다운로드

  1. [검색] 버튼 클릭
  2. 다운로드 영역 → "전체읍면동현황" 라디오 버튼 선택
  3. [CSV 파일 다운로드] 클릭 → 알림창 확인 → 10~30초 대기

④ 같은 폴더에 저장 후 열기.command 실행

CSV 파일을 이 HTML과 같은 폴더에 넣고 열기.command를 더블클릭하면 자동으로 로드됩니다.
(열기.command 없이 HTML을 직접 열면 아래 업로드 영역에 드래그앤드롭)

⑤ 예상 파일 규격

파일명: YYYYMM_YYYYMM_연령별인구현황_연간.csv · 인코딩: EUC-KR · 컬럼 수: 208개 · 행 수: ~4,200행
